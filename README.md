# ОПИСАНИЕ ПРОДУКТА ИВР

«Моделирование квантового туннельного эффекта»

## 1. Что представляет собой продукт

Продукт - интерактивная программа-симулятор для изучения квантового туннельного эффекта. Реализована в виде одного HTML-файла.

## 2. Функциональные составляющие

1. **Слайдеры параметров:** энергия частицы E (0,1–9,9 эВ), масса m (0,1–5,0 mₑ), высота барьера V₀ (0,5–15 эВ), ширина барьера d (0,1–3,0 нм).
2. **Два режима барьера:** прямоугольный (аналитический расчёт) и произвольный (форма рисуется мышью, расчёт — метод матриц переноса).
3. **Три вкладки:** волновая функция |ψ(x)|², зависимость T(E), верификация по данным СТМ.
4. **15 встроенных экспериментальных СТМ-измерений;** поддерживается загрузка собственных данных в формате CSV.
5. **Панель метрик:** коэффициент туннелирования T, затухание κ, режим, отношение E/V₀.
6. **Четыре пресета** для быстрого перехода к типовым физическим сценариям.

## 3. Физическая модель

Расчёт ведётся точным методом матриц переноса. Используемые формулы:

1. **E < V₀ (туннельный режим):**
  T = [1 + V₀² sinh²(κd) / (4E(V₀−E))]⁻¹

2. **E ≥ V₀ (надбарьерное прохождение):**
  T = [1 + (k₁²−k₂²)² sin²(k₂d) / (4k₁²k₂²)]⁻¹

3. **Коэффициент затухания:**
  κ = √(2m(V₀−E)) / ℏ

## 4. Связь с задачами проекта

1. **Численное решение уравнения Шрёдингера** - метод матриц переноса.
2. **Визуализация волновой функции и T** - интерактивные графики.
3. **Изменение параметров в реальном времени** - слайдеры.
- **Сравнение с данными СТМ** — вкладка «Верификация СТМ».
- **Доступность без установки ПО** — HTML и JavaScript.
