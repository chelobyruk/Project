# ОПИСАНИЕ ПРОДУКТА ИВР

«Моделирование квантового туннельного эффекта»

## 1. Что представляет собой продукт

Продукт — интерактивная программа-симулятор для изучения квантового туннельного эффекта. Реализована в виде одного HTML-файла.

## 2. Функциональные составляющие

- **Слайдеры параметров:** энергия частицы E (0,1–9,9 эВ), масса m (0,1–5,0 mₑ), высота барьера V₀ (0,5–15 эВ), ширина барьера d (0,1–3,0 нм).
- **Два режима барьера:** прямоугольный (аналитический расчёт) и произвольный (форма рисуется мышью, расчёт — метод матриц переноса).
- **Три вкладки:** волновая функция |ψ(x)|², зависимость T(E), верификация по данным СТМ.
- **15 встроенных экспериментальных СТМ-измерений;** поддерживается загрузка собственных данных в формате CSV.
- **Панель метрик:** коэффициент туннелирования T, затухание κ, режим, отношение E/V₀.
- **Четыре пресета** для быстрого перехода к типовым физическим сценариям.

## 3. Физическая модель

Расчёт ведётся точным методом матриц переноса. Используемые формулы:

- **E < V₀ (туннельный режим):**
  T = [1 + V₀² sinh²(κd) / (4E(V₀−E))]⁻¹

- **E ≥ V₀ (надбарьерное прохождение):**
  T = [1 + (k₁²−k₂²)² sin²(k₂d) / (4k₁²k₂²)]⁻¹

- **Коэффициент затухания:**
  κ = √(2m(V₀−E)) / ℏ

## 4. Связь с задачами проекта

- **Численное решение уравнения Шрёдингера** — метод матриц переноса.
- **Визуализация волновой функции и T** — интерактивные графики.
- **Изменение параметров в реальном времени** — слайдеры.
- **Сравнение с данными СТМ** — вкладка «Верификация СТМ».
- **Доступность без установки ПО** — HTML и JavaScript.
